# dumbfile
Generate 1GByte dumb file: 
```
root@srv-flow:~# ./dumbfile.sh 1
File of 1 GBytes aproximated. Generating...
20% concluded...
40% concluded...
60% concluded...
80% concluded...
100% concluded!
```
Your dumb file is located at same place of dumbfile.sh script! **Please input only integers!!!**
